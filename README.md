# Responsive React Navbar

## Create React App

```bash
npx create-react-app react-navbar
```

Install react-router-dom to use `<Link>`

```bash
npm install react-router-dom
```

```bash
npm start
```

## Add Font Awesome

Add link to CDN in `public/index.html`

## Add Google Fonts

Add link in `public/index.html`
